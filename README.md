# 📊 个人记账管理系统

> 基于 React 开发的移动端个人记账应用，支持收支记录、年度数据统计与可视化展示。

## ✨ 功能亮点
- 💰 日常收支记录与分类管理
- 📈 年度账单统计：总收入/总支出/总结余
- 📊 月度收支趋势折线图 + 支出分类占比饼图
- 📱 移动端响应式布局，适配手机端使用

## 🛠️ 技术栈
- 框架：React 19 + Redux Toolkit
- UI 组件库：Ant Design Mobile
- 数据可视化：Recharts
- 日期处理：Day.js
- 样式：SCSS
- 工具库：Lodash

# 安装依赖
